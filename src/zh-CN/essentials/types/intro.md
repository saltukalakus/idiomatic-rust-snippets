### 类型

Rust 中的类型定义了程序中可以存储和操作的数据种类。Rust 是静态类型语言，这意味着每个变量的类型在编译时必须已知。这有助于在开发早期捕获许多错误并保证内存安全。

1- [标量类型](./scalar.md)：包括整数、浮点数、布尔值和字符等单值类型。

2- [复合类型](./compound.md)：将多个值组合为一个的类型，包括元组和数组。

3- [结构体](./struct.md)：允许将相关数据和方法组合在一起的自定义数据类型。

4- [枚举](./enum.md)：列举可能值的类型，每个变体可以可选地包含关联数据。

5- [特质（Trait）](./trait.md)：以抽象方式定义共享行为，类似于其他语言中的接口。

6- [泛型](./generic.md)：编写灵活且可重用的代码，可在不牺牲性能的前提下适用于多种类型。

7- [关联类型](./associated-types.md)：在特质中关联类型占位符，以获得更简洁的 API 和更好的类型推断。

8- [指针](./pointer.md)：表示内存地址的类型，用于引用内存中存储的数据，同时保持 Rust 的安全保证。

9- [向量（Vec）](./vector.md)：用于动态集合的可增长数组类型，在运行时可增长或缩小。

10- [类型别名](./type-alias.md)：为现有类型创建替代名称以提高代码可读性。

11- [PhantomData](./phantom-data.md)：用于编译期类型检查和类型状态模式的零大小标记类型。

12- [引用 与 智能指针](./reference-vs-smart-pointer.md)：理解普通引用与智能指针之间的差异。

13- [结构体 与 枚举](./struct-vs-enum.md)：何时在 Rust 代码中使用结构体与枚举。
