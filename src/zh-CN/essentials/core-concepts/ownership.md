### 所有权

所有权是一组规则，用来控制程序如何管理内存。所有权系统旨在防止内存泄漏并保证内存安全。以下是所有权的关键规则：

1. 每个值都有一个称为其所有者的变量。
2. 同一时间只能有一个所有者。
3. 当所有者超出作用域时，该值将被释放（drop）。

```rust, editable
fn main() {
    let s1 = String::from("hello");
    let s2 = s1; // s1 被移动到 s2

    // println!("{}", s1); // 这一行会导致编译错误，因为 s1 不再有效
    println!("{}", s2); // s2 现在是该字符串的所有者
}
```
