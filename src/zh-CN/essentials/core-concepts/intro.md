### 核心概念

Rust 有若干核心概念，它们是理解并高效使用这门语言的基础。这些概念旨在在不牺牲易用性的前提下，保证内存安全、并发性和高性能。

1- [所有权](./ownership.md)：一组规则，控制 Rust 程序如何管理内存，从而在无需垃圾回收器的情况下保证内存安全。

2- [借用](./borrow.md)：允许引用一个值而不取得其所有权。

3- [生命周期](./lifetime.md)：用于确保引用在使用期间有效，防止出现悬垂引用，从而保证内存安全。

4- [迭代器](./iterator.md)：使用 `Iterator` 特质以惰性求值和零成本抽象高效地处理元素序列。

5- [流程控制](./flow-control.md)：用于控制程序执行的循环与条件语句等控制流机制。

6- [模式匹配](./pattern-matching.md)：强大的功能，允许匹配复杂数据结构并根据其形状执行相应代码。

7- [并发](./concurrency.md)：提供线程与通道等强大的并发原语，同时通过所有权与借用保证内存安全。

8- [异步/Await](./async-await.md)：使用 `async/await` 语法进行异步编程，针对 I/O 密集型操作实现高效非阻塞。

9- [错误处理](./error-handling.md)：使用 `Result` 与 `Option` 枚举的错误处理体系，鼓励显式处理错误而非异常。

10- [宏](./macro.md)：用于编写生成代码的元编程工具，减少样板代码并生成代码片段。

11- [模块](./module.md)：Rust 用于组织代码、将相关代码分组的机制。

12- [Crate（包）](./crate.md)：Rust 中的编译与分发单元。

13- [函数](./function.md)：组织与重用代码的基础构建块，用于封装逻辑并执行特定任务。

14- [闭包](./closure.md)：可以从其环境中捕获值的匿名函数。
