### 算术运算

算术运算符用于执行基本的数学运算。

`+` : 加法<br/>
`-` : 减法<br/>
`*` : 乘法<br/>
`/` : 除法<br/>
`%` : 余数（取模）<br/>

```rust, editable
fn main() {
    let a = 10;
    let b = 3;

    println!("a + b = {}", a + b); // 加法
    println!("a - b = {}", a - b); // 减法
    println!("a * b = {}", a * b); // 乘法
    println!("a / b = {}", a / b); // 除法
    println!("a % b = {}", a % b); // 余数
}
```
