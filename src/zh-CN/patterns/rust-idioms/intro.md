### Rust 习惯用法

这些模式是 Rust 独有的，或者具有充分利用语言所有权系统、类型系统和零成本抽象的 Rust 特定实现。它们常用于地道的 Rust 代码，可能在其他语言中没有直接对应物。

**关键特征**：
- 充分利用 Rust 的所有权和借用系统
- 提供编译期保证
- 通常无运行时成本
- 利用 Rust 强大的类型系统

### Rust 特定模式

1- [Newtype 模式](./newtype.md)：包装类型以实现类型安全和抽象，无运行时开销。

2- [RAII（资源获取即初始化）](./raii.md)：使用 Rust 的所有权和 Drop trait 实现自动资源管理。

3- [扩展 Trait](./extension-traits.md)：通过 trait 实现为不属于你的类型添加方法。

4- [类型状态模式](./typestate.md)：在类型系统中编码状态机，实现编译期状态验证。
