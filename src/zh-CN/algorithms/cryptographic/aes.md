### AES 算法

AES（高级加密标准）是一种广泛使用的对称加密算法，用于保护数据。它以 128 位为固定分块大小，并支持 128、192 或 256 位密钥。本示例演示一个简化的教学性 AES-128 加密（仅加密，不含解密）。

**注意**：这是一个简化的教学实现。用于生产环境时，请始终使用 `aes` crate 或其他经过充分测试的密码学库。

```rust, editable 
fn sub_bytes(state: &mut [u8; 16]) {
    for i in 0..16 {
        state[i] = S_BOX[state[i] as usize];
    }
}

fn shift_rows(state: &mut [u8; 16]) {
    let temp = *state;
    // Row 0: no shift
    // Row 1: shift left by 1
    state[1] = temp[5];
    state[5] = temp[9];
    state[9] = temp[13];
    state[13] = temp[1];
    // Row 2: shift left by 2
    state[2] = temp[10];
    state[6] = temp[14];
    state[10] = temp[2];
    state[14] = temp[6];
    // Row 3: shift left by 3
    state[3] = temp[15];
    state[7] = temp[3];
    state[11] = temp[7];
    state[15] = temp[11];
}

fn xtime(x: u8) -> u8 {
    (x << 1) ^ (((x >> 7) & 1) * 0x1b)
}

fn mix_columns(state: &mut [u8; 16]) {
    for i in 0..4 {
        let s0 = state[i * 4];
        let s1 = state[i * 4 + 1];
        let s2 = state[i * 4 + 2];
        let s3 = state[i * 4 + 3];
        
        state[i * 4] = xtime(s0) ^ xtime(s1) ^ s1 ^ s2 ^ s3;
        state[i * 4 + 1] = s0 ^ xtime(s1) ^ xtime(s2) ^ s2 ^ s3;
        state[i * 4 + 2] = s0 ^ s1 ^ xtime(s2) ^ xtime(s3) ^ s3;
        state[i * 4 + 3] = xtime(s0) ^ s0 ^ s1 ^ s2 ^ xtime(s3);
    }
}

fn add_round_key(state: &mut [u8; 16], round_key: &[u8]) {
    for i in 0..16 {
        state[i] ^= round_key[i];
    }
}

fn key_expansion(key: &[u8; 16], round_keys: &mut [u8; 176]) {
    const RCON: [u8; 10] = [0x01, 0x02, 0x04, 0x08, 0x10, 0x20, 0x40, 0x80, 0x1b, 0x36];
    
    // Copy original key
    round_keys[0..16].copy_from_slice(key);
    
    for i in 4..44 {
        let mut temp = [
            round_keys[(i - 1) * 4],
            round_keys[(i - 1) * 4 + 1],
            round_keys[(i - 1) * 4 + 2],
            round_keys[(i - 1) * 4 + 3],
        ];
        
        if i % 4 == 0 {
            // RotWord
            temp.rotate_left(1);
            // SubWord
            for j in 0..4 {
                temp[j] = S_BOX[temp[j] as usize];
            }
            // XOR with Rcon
            temp[0] ^= RCON[i / 4 - 1];
        }
        
        for j in 0..4 {
            round_keys[i * 4 + j] = round_keys[(i - 4) * 4 + j] ^ temp[j];
        }
    }
}

fn aes_encrypt_block(block: &mut [u8; 16], key: &[u8; 16]) {
    let mut round_keys = [0u8; 176];
    key_expansion(key, &mut round_keys);

    add_round_key(block, &round_keys[0..16]);

    for round in 1..10 {
        sub_bytes(block);
        shift_rows(block);
        mix_columns(block);
        add_round_key(block, &round_keys[round * 16..(round + 1) * 16]);
    }

    // Final round (no MixColumns)
    sub_bytes(block);
    shift_rows(block);
    add_round_key(block, &round_keys[160..176]);
}

fn main() {
    let mut block = [0x32, 0x43, 0xf6, 0xa8, 0x88, 0x5a, 0x30, 0x8d, 
                     0x31, 0x31, 0x98, 0xa2, 0xe0, 0x37, 0x07, 0x34];
    let key = [0x2b, 0x7e, 0x15, 0x16, 0x28, 0xae, 0xd2, 0xa6, 
               0xab, 0xf7, 0x4d, 0x4a, 0x4e, 0xa6, 0x2b, 0x7e];

    println!("Original block: {:02x?}", block);
    aes_encrypt_block(&mut block, &key);
    println!("Encrypted block: {:02x?}", block);
}
```

- **SubBytes**: 将状态中的每个字节替换为 S-Box 中对应的字节，提供非线性。S-Box 是一个 256 字节的查找表。
- **ShiftRows**: 循环左移状态数组的行，第 i 行左移 i 个位置（第 0 行不移动，第 1 行移动 1 位，第 2 行移动 2 位，第 3 行移动 3 位）。
- **MixColumns**: 使用有限域（GF(2^8)）中的乘法混合每列的数据，在整个分组中提供扩散。`xtime` 函数实现 GF(2^8) 中乘以 2 的操作。
- **AddRoundKey**: 将状态与从原始密钥派生出的轮密钥进行异或（XOR）。
- **KeyExpansion**: 从初始 16 字节密钥生成 11 个轮密钥（共 176 字节），使用 RotWord（旋转）、SubWord（S-Box 替换）和 Rcon（轮常量）。
- `aes_encrypt_block` 函数对 10 轮（AES-128）进行加密，每轮应用 SubBytes、ShiftRows、MixColumns 和 AddRoundKey。
- 最后一轮根据 AES 规范省略了 MixColumns 步骤。
- 该示例实现为一个完整的、简化的教学性 AES-128 加密实现。它演示了所有核心变换、S-Box、有限域运算和密钥调度。
