### SHA-256 散列算法

SHA-256（安全哈希算法 256 位）是一种密码学哈希函数，它可以将任意长度的输入映射为固定的 256 位（32 字节）哈希值。它被广泛用于数据完整性校验、数字签名和密码存储。

```rust, editable
const K: [u32; 64] = [
    0x428a2f98, 0x71374491, 0xb5c0fbcf, 0xe9b5dba5, 0x3956c25b, 0x59f111f1, 0x923f82a4, 0xab1c5ed5,

fn sha256(input: &[u8]) -> [u8; 32] {
    let mut h: [u32; 8] = [
        0x6a09e667, 0xbb67ae85, 0x3c6ef372, 0xa54ff53a,
        0x510e527f, 0x9b05688c, 0x1f83d9ab, 0x5be0cd19,
    ];

    let mut padded = input.to_vec();
    let bit_len = (input.len() as u64) * 8;
    padded.push(0x80);
    while (padded.len() % 64) != 56 {
        padded.push(0);
    }
    padded.extend_from_slice(&bit_len.to_be_bytes());

    for chunk in padded.chunks(64) {
        let mut w = [0u32; 64];
        for i in 0..16 {
            w[i] = u32::from_be_bytes([chunk[4 * i], chunk[4 * i + 1], chunk[4 * i + 2], chunk[4 * i + 3]]);
        }
        for i in 16..64 {
            let s0 = w[i - 15].rotate_right(7) ^ w[i - 15].rotate_right(18) ^ (w[i - 15] >> 3);
            let s1 = w[i - 2].rotate_right(17) ^ w[i - 2].rotate_right(19) ^ (w[i - 2] >> 10);
            w[i] = w[i - 16].wrapping_add(s0).wrapping_add(w[i - 7]).wrapping_add(s1);
        }

        let mut a = h[0];
        let mut b = h[1];
        let mut c = h[2];
        let mut d = h[3];
        let mut e = h[4];
        let mut f = h[5];
        let mut g = h[6];
        let mut h_var = h[7];

        for i in 0..64 {
            let s1 = e.rotate_right(6) ^ e.rotate_right(11) ^ e.rotate_right(25);
            let ch = (e & f) ^ (!e & g);
            let temp1 = h_var.wrapping_add(s1).wrapping_add(ch).wrapping_add(K[i]).wrapping_add(w[i]);
            let s0 = a.rotate_right(2) ^ a.rotate_right(13) ^ a.rotate_right(22);
            let maj = (a & b) ^ (a & c) ^ (b & c);
            let temp2 = s0.wrapping_add(maj);

            h_var = g;
            g = f;
            f = e;
            e = d.wrapping_add(temp1);
            d = c;
            c = b;
            b = a;
            a = temp1.wrapping_add(temp2);
        }

        h[0] = h[0].wrapping_add(a);
        h[1] = h[1].wrapping_add(b);
        h[2] = h[2].wrapping_add(c);
        h[3] = h[3].wrapping_add(d);
        h[4] = h[4].wrapping_add(e);
        h[5] = h[5].wrapping_add(f);
        h[6] = h[6].wrapping_add(g);
        h[7] = h[7].wrapping_add(h_var);
    }

    let mut result = [0u8; 32];
    for (i, &val) in h.iter().enumerate() {
        result[4 * i..4 * i + 4].copy_from_slice(&val.to_be_bytes());
    }
    result
}

fn main() {
    let data = b"hello world";
    let hash = sha256(data);
    for byte in &hash {
        print!("{:02x}", byte);
    }
    println!();
}
```

- 常量数组 `K` 包含 64 个轮常量，来源于前 64 个素数立方根的小数部分。
- 初始哈希值 `h` 来源于前 8 个素数平方根的小数部分。
- 输入通过在末尾追加一个 '1' 位、若干 '0' 位以及原始消息长度（以位为单位）进行填充。
- 填充确保最终消息长度为 512 位（64 字节）的倍数。
- 填充后的消息以 512 位（64 字节）块为单位处理。
- 对于每个分块，构造长度为 64 的消息调度数组，前 16 个字直接取自分块，其余 48 个字通过位运算计算得到。
- 主压缩循环执行 64 轮位运算、模加和逻辑函数对工作变量 a 到 h 进行处理。
- 处理完所有分块后，将哈希值拼接得到最终的 256 位哈希。
- `main` 函数演示对 "hello world" 的哈希，结果为：`b94d27b9934d3e08a52e52d7da7dabfac484efe37a5380ee9088f7ace2efcde9`。
- 本实现用于教学目的；用于生产环境请使用 `sha2` crate 或其他经过充分测试的密码学库。
