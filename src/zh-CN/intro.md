### 惯用 Rust 代码片段

Rust 众所周知有较陡峭的学习曲线。本书旨在对初学者友好，帮助你快速掌握 Rust 的核心概念，同时作为 Rust 开发者的实用速查手册。

1. [基础](./essentials/intro.md) 包含 Rust 语言的基本方面，通过简洁易懂的示例进行讲解。

2. [反模式](./anti-patterns/intro.md) 部分解释需要避免的非惯用写法。对于已经开始构建项目的开发者，这是最值得阅读的部分。

3. [设计模式](./patterns/intro.md) 部分展示了若干设计模式，使用极简但完整的示例项目，适合更高级的开发者与架构师。

4. [算法](./algorithms/intro.md) 部分使用 Rust 展示若干常用算法的示例代码。

本项目采取极简风格，大多数页面都很短且包含简单易懂的代码示例。大部分示例可以运行。对于允许在线执行的示例，你可以在在线编辑器中修改代码并实时测试你的想法。

如需更多信息，必要时会引用外部文档。
